# Z80oolong Tts

## How do I install these formulae?

`brew install z80oolong/tts/<formula>`

Or `brew tap z80oolong/tts` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "z80oolong/tts"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
